KinFin
===============================

version number: 0.9
author: Dominik R Laetsch

Overview
--------

Taxon-aware analysis of clustered protein data

Installation / Usage
--------------------

To install use pip:

    $ pip install kinfin


Or clone the repo:

    $ git clone https://github.com/DRL/kinfin.git
    $ python setup.py install
