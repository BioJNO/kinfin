Dependencies
------------
* bash
* wget
* gunzip
* Python2.7+
* pip

Installation
------------

    $ git clone https://github.com/DRL/kinfin.git
    $ cd kinfin
    $ ./install

Usage
-----

    $ ./kinfin -h

Example
-------

    $ ./test
